# Clippizo Storybook

UI component library for the Clippizo AI video creation platform.

## Getting Started

```bash
npm run dev --filter storybook
```

Open [http://localhost:6006](http://localhost:6006) with your browser to see the component library.

## About

This Storybook instance showcases all UI components used in the Clippizo platform, including:

- Design system components
- Form elements
- Layout components
- AI-specific UI elements (generation progress, asset previews, etc.)
